📖 О проекте

Этот блог создан в рамках учебного задания по теме "View и URLS. Представление и маршруты" курса Python-разработчик.

Проект демонстрирует работу с маршрутизацией Django, представлениями на основе классов (Class-Based Views) и формами с валидацией данных.

🎯 Цели задания

  • ✅ Создавать простые и динамические маршруты
  • ✅ Связывать маршруты с представлениями и шаблонами
  • ✅ Работать с формами и обрабатывать данные запросов
  • ✅ Реализовывать полный CRUD (создание, чтение, обновление, удаление)

🛠️ Что реализовано

🔗 Маршруты (URLs)
  • / — Главная страница
  • /about/ — О нас
  • /posts/ — Список постов
  • /posts/<id>/ — Детальный просмотр
  • /posts/create/ — Создание поста
  • /posts/<id>/update/ — Редактирование
  • /posts/<id>/delete/ — Удаление
👁️ Представления (Views)
  • home, about — функции
  • PostListView — список
  • PostDetailView — детали
  • PostCreateView — создание
  • PostUpdateView — обновление
  • PostDeleteView — удаление

⚙️ Технологии

Python 3.10 Django 5.2 HTML5 CSS3 Bootstrap 5 SQLite

🧪 Тестирование

Проект покрыт автоматическими тестами Django TestCase:

python manage.py test
